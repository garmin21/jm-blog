scroll-behavior 平滑滚动

约 249 字小于 1 分钟

CSS

2024-07-10

scroll-behavior

scroll-behavior:smooth 写在滚动容器元素上，可以让容器（非鼠标手势触发）的滚动变得平滑。

语法

scroll-behavior: auto;
scroll-behavior: smooth;
.tab {
  overflow: hidden;
}
.label {
  width: 100px;
  margin-right: -1px;
  border: 1px solid #ccc;
  border-bottom: 0;
  padding-top: 5px;
  padding-bottom: 5px;
  background-color: #eee;
  text-align: center;
  float: left;
}
.box {
  height: 200px;
  border: 1px solid #ccc;
  /* ======= */
  scroll-behavior: smooth;
  /* ======= */
  overflow: hidden;
}
.content {
  height: 100%;
  padding: 0 20px;
  position: relative;
  overflow: hidden;
}
.box input {
  position: absolute;
  top: 0;
  height: 100%;
  width: 1px;
  border: 0;
  padding: 0;
  margin: 0;
  clip: rect(0 0 0 0);
}

</div>
